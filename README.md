Небольшой одностраничный сайт-лендинг, демонстрирующий услуги сервиса с динамической загрузкой данных.  
Проект делался на заказ для шиномонтажной сети 5Колесо.

Функциональность

- Отображение промо-акции
- Загрузка данных из JSON:
  - цены (`prices.json`)
  - локации (`locations.json`)
  - список объектов (`object_list.js`)
- Рендеринг элементов на странице через JavaScript
- Адаптивная верстка под мобильные устройства
